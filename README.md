# ❤️ Heart Disease Prediction using Machine Learning

##  Overview

This project uses machine learning to predict whether a patient has heart disease based on medical features. The model is built using Logistic Regression and includes data analysis and visualization.

---

## Objective

The goal of this project is to build a predictive model that can classify patients into:

* 0 → No Heart Disease
* 1 → Heart Disease

---

## 📊 Dataset

The dataset contains medical information about patients, including:

* Age
* Sex
* Chest pain type
* Blood pressure
* Cholesterol level
* And other health indicators

---
